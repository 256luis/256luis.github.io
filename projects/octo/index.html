Octo

Octo is a modern, statically and strongly-typed programming language that transpiles to C code. It offers a clean syntax reminiscent of other high-level languages, supporting features like basic type inference, arrays, pointers, and user-defined structs and unions. The language is strict with types, disallowing implicit casts between signed and unsigned integers or between numeric and floating-point types, and only allowing implicit casting from smaller to larger numeric types. Octo also provides familiar control flow structures like if-statements, while-loops, and Python-like for-loops for iterating over arrays.

extern func printf(format: &char, ..) -> i32;

func fib(n: i64) -> i64
{
    if n == 0 return 0;
    if n == 1 return 1;

    return fib(n - 1) + fib(n - 2);
}

func main() -> i32
{
    let n = 5;
    let nth_fib = fib(n);
    printf("fib(%d) = %d", n, nth_fib);
}
